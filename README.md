# Hi! It's my first react desktop app - _Employees App_

This project was bootstrapped with
[Create React App](https://github.com/facebook/create-react-app).

## - About this App

### In this mini app, we can:

- Add new employees in the form below. The minimum number of characters is 3
  symbols, the default salary value is $1000,
- Filter employees by 3 buttons: all, increase, salary more than $1000,
- Remove employees,
- Add increase for employees by clicking on a cookie,
- Highlight the best employees by clicking on the employee,
- Remove employees by clicking on the basket,
- Employee statistics are in the header,
- Local storage and adaptive will be added in the near future.
